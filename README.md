Investment Return Calculator

An Interactive Excel Financial Comparison Tool

By Triet Nguyen

This Excel-based investment return calculator allows users to compare two investment scenarios side by side. By adjusting initial investment, monthly contributions, return rate, and time horizon, users can instantly see how different strategies affect long-term wealth building.

Features
- Two scenario comparison — compare aggressive vs conservative investment strategies side by side
- Dynamic inputs — change any variable and all calculations update automatically
- Year by year growth table — shows balance for both scenarios across the full investment period
- Line chart visualization — clearly shows the compounding gap between scenarios over time
- Future Value calculation — uses Excel's FV function for accurate compound interest math


How to Use
1) Download the Investment_Return_Calculator.xlsx file
2) Open it in Microsoft Excel or Google Sheets
3) Enter your values in the input sections:
- Initial Investment — starting amount
- Monthly Contribution — amount added each month
- Annual Return Rate — as a decimal (e.g. 0.07 for 7%)
- Years — investment time horizon
4) The Future Value, year by year table, and chart all update automatically
5) Adjust Scenario 2 inputs to compare a different strategy


Formulas Used
- FV — calculates future value of an investment with regular contributions
- Absolute cell references ($B$4) to ensure formulas calculate correctly across both scenarios
- Dynamic year by year table auto-populates using relative references
